​																	Docker 설치 방법



1. brew를 사용해서 설치

```
brew install docker
```

2. 설치가 잘 되었는지 확인

```
docker version
```

3. 위의 것은 client 서버를 설치해보자.

```
https://docs.docker.com/docker-for-mac/install/
```

4. 다시 docker version을 하면 server가 나온다

```
docker version
```
